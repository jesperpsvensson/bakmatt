Om Bakmått

Bakmått är en liten webbplikation, gjord med HTML, CSS, Bootstrap, Javascript samt en liten gnutta Jquery. Med Bakmått kan man enkelt överätta mått och enheter som kan förekomma i köket och framförallt vid bakning.

Happy baking!
